# 팀원 프로필 사진 가이드

## 📁 이미지 업로드 위치
이 폴더(`/public/images/team/`)에 팀원 프로필 사진을 업로드하세요.

## 📸 이미지 권장 사양
- **형식**: JPG, PNG, WebP
- **크기**: 800x800px 이상 (정사각형 권장)
- **용량**: 500KB 이하 권장
- **비율**: 1:1 (정사각형) 또는 3:4 (세로)
- **품질**: 고해상도, 선명한 얼굴

## 📝 파일명 규칙
파일명은 자유롭게 지정 가능합니다. 예:
- `joosik-choi.jpg`
- `kai-kim.png`
- `martin-hyun.webp`

## 🔧 사용 방법

### 1. 이미지 파일을 이 폴더에 업로드
```
public/images/team/
├── joosik-choi.jpg
├── kai-kim.jpg
├── martin-hyun.jpg
└── ...
```

### 2. content.data.json에서 이미지 경로 수정
```json
{
  "team": {
    "members": [
      {
        "id": 1,
        "name": "Joosik Choi",
        "image": "/images/team/joosik-choi.jpg"
      }
    ]
  }
}
```

### 3. 콘텐츠 재생성
```bash
npm run content:gen
```

## 💡 참고사항
- `/public` 폴더의 파일은 빌드 시 그대로 복사됩니다
- 경로는 항상 `/images/team/파일명.확장자` 형식으로 시작합니다
- 이미지가 없으면 자동으로 기본 아이콘이 표시됩니다
- 이미지 로드 실패 시에도 폴백 아이콘이 표시됩니다

## 🎨 이미지 최적화 팁
1. **배경 제거**: 전문적인 느낌을 위해 배경을 단색이나 투명으로 설정
2. **조명**: 얼굴이 잘 보이도록 밝고 균일한 조명
3. **구도**: 얼굴이 중앙에 위치하고 적절한 여백 확보
4. **일관성**: 모든 팀원 사진의 스타일을 일관되게 유지
